Instale todos os arquivos para que funcione corretamente, porem caso possivel baixe da aba de releases para mais facilidade

Para executar este programa corretamente, primeiro execute o arquivo de registro, copie sua senha e depois prossiga para o login.

Certifique-se de ter o Python instalado, além da biblioteca hashlib e das demais dependências necessárias para que o programa funcione.

Execute os arquivos pelo Prompt de Comando (CMD):

Clique com o botão direito na pasta onde os arquivos estão localizados.
Selecione "Abrir no Terminal" ou "Abrir janela de comando aqui".
Digite o seguinte comando:
python registration.py

Após concluir o registro, execute:

python login.py

Siga as instruções exibidas na tela para finalizar o processo de login.
